# Task Queue

Processed automatically by `scripts/run-tasks.sh`.
Format: `* [ ] TASK: description` — one task per line.
Script picks the **first** uncompleted task, runs `claude -p`, marks done, then picks the next.

Status legend: `[ ]` pending · `[*]` in progress · `[x]` done · `[!]` failed
